Features

Pomodoro Timer

Customisable focus and break intervals. Stays accurate across background tabs and device sleep using wall-clock syncing.

Live Leaderboard

Weekly and all-time rankings filtered by university. Users can opt out with a built-in incognito mode.

Unit Breakdown

A weekly bar chart showing time spent per university unit — makes it easy to spot neglected subjects before it's too late.

Daily Streak

Tracks consecutive study days to build consistent habits and motivate daily engagement.

Live Presence

Real-time count of users currently studying, powered by Supabase Realtime presence channels.

Task Manager

Create tasks tagged to units with optional due dates. Tasks sort automatically by due date and are linked to study sessions.

Technical Challenges

Timer Drift & Background Tab Accuracy

Browsers throttle setInterval in background tabs, causing the countdown to drift. Solved by recording a wall-clock start time and recalculating elapsed time on every tick, then re-syncing on visibility change so the timer stays accurate even after the laptop sleeps.

Row-Level Security & Data Privacy

A security review found that without Postgres RLS policies, authenticated users could query any other user's data. Implemented per-table RLS policies and a public_profiles view that deliberately omits sensitive columns, ensuring users can only access what they're permitted to see.

Leaderboard Performance

The leaderboard aggregates session data across all users. Structured queries to fetch totals first then batch-resolve display names in a single request, avoiding N+1 queries and keeping load times fast as the user base grows.
